POS Save Quotation

A POS Save Quotation is a module which allow POS user to save order as quotation which can be loaded later. It provides a special feature such that the seller can create quotation for any order in running POS session and load this quotation when required. It allows the seller to save an order as quotation in the running POS and reload any order from the list of saved orders.

Seller can save quotation from POS Session to backend as well as seller can load quotation to POS Session from backend.

On clicking 'Save Quotation' button, this window will be shown. Seller can save quotation here as well as print quotation details. On clicking 'Print' button, quotation will be saved at backend and quotation receipt will be generated automatically.

On clicking 'Load Quotation' button, this window will be shown. Seller can search and select quotation from the list of quotations that he wishes to load.

After Selecting the quote id respected quotation details will Show in popup
